# 🛍️ Shopsmart API

Shopsmart is a FastAPI-based e-commerce backend project built with **FastAPI**, **SQLAlchemy**, and **MySQL/SQLite**.  
It provides REST APIs for managing **Customers**, **Products**, and **Transactions**, along with a base structure for future machine learning–powered product recommendations.

---

## 🚀 Features
- Customer Management (CRUD)
- Product Management (CRUD)
- Transaction Management (CRUD)
- Organized modular project structure
- Database support (MySQL / SQLite)
- Ready for ML-based recommendation system
